## 🚀 Deployment

### GitHub Pages Setup

1. **Create a repository** named `himanshugupta08.github.io`
2. **Push files** to main branch:
   ```bash
   git add .
   git commit -m "Initial portfolio"
   git push -u origin main
   ```
3. **Enable GitHub Pages** (Settings → Pages → Source: main branch)
4. **Visit:** `https://himanshugupta08.github.io`

### Custom Domain (Optional)
- Add `CNAME` file with your domain
- Configure DNS records at your registrar

---

## 🎨 Customization

### Change Colors
Edit CSS variables in `<style>`:
```css
:root {
  --accent: #00d4aa;        /* Primary color */
  --accent2: #0080ff;       /* Secondary */
  --bg: #0a0e17;           /* Background */
  --text: #e2e8f0;         /* Text */
}
```

### Update Profile Picture
Replace `assets/myPic.webp` with your image (maintain 320×400px for best fit)

### Edit Content
- Modify HTML directly in `index.html`
- Update skills, experience, projects, certifications
- Change social links and contact info
